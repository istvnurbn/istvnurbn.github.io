## Building a similar site

1. [Install Hugo](https://gohugo.io/getting-started/installing/)
2. Get the [Congo theme](https://github.com/jpanther/congo)
3. Check the `/config/_default` folder for configuration and the contents of the `/content` folder for inspiration.
4. Use GitHub Actions and GitHub Pages to publish/host your site.
