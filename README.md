## Building a similar site

1. [Install Hugo](https://gohugo.io/getting-started/installing/)
2. Get the [WonderMod theme](https://github.com/Wonderfall/hugo-WonderMod)
3. Check the `config.yml` file and the contents of the aptly named `/content` folder for inspiration.
4. Use GitHub Actions and GitHub Pages to publish/host your site.
